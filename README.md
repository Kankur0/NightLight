# NightLight
This Stardew Valley mod allows the player to adjust the darkness of night in various areas where it becomes dark such as outdoors, the mines (most notable on floors 30-39 of the regular mines), the farm cave, the farmhouse, etc.

## Requirements:
- Stardew Valley 1.6+
- [SMAPI 4.0.0+](https://smapi.io/)

## Optional:
- [Generic Mod Configuration Menu](http://www.nexusmods.com/stardewvalley/mods/5098)
  - You can use the mod without, but you will have to manually edit the config.json file if so.
